# Football Plays Simulation System

A Python-based project that designs and simulates offensive flag football plays using the Turtle graphics library.

---

## Overview

This project contains 11 custom-designed offensive plays implemented as independent Python modules.

Each play visually simulates:
- Player positioning
- Route movement
- Execution timing

The objective was to translate real-world football strategy into structured, executable code while maintaining modularity and clarity in system design.

---

## Features

- 11 unique offensive play simulations
- Visual rendering of player routes using Turtle graphics
- Modular architecture (one `.py` file per play)
- Clear separation of play logic and rendering logic
- Recorded demonstrations explaining play strategy and execution

---

## Tech Stack

- Python 3
- Turtle Graphics Library
- Modular scripting structure

---

## How It Works

Each play script:

- Initializes player alignment
- Defines route paths
- Controls movement timing
- Animates execution using Turtle graphics

The simulation visually demonstrates spacing, sequencing, and strategic objectives for each designed play.

---

## How to Run

1. Ensure Python 3 is installed.

2. Clone the repository:

git clone https://github.com/yourusername/Football-Play-Designs.git

3. Navigate to the project directory:

cd Football-Play-Designs

4. Run any play:
   
- press play(top right button) of file
  OR
- python3 Bunch.py in command line(no need to open file)
